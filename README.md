# firebase-first-demo

[URL:https://first-demo-a6d75.web.app/](https://first-demo-a6d75.web.app/)

## Google へのログイン

```
firebase login
```

### Git Bash では --interactive というオプションを付けないと対話型モードにならない

```
firebase login --interactive
```

## プロジェクトの開始

```
firebase init
```

## ウェブアプリをデプロイ

```
firebase deploy
```
